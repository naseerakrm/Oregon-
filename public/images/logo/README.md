# Orecoin Branding Assets

This directory contains the official branding assets for Orecoin.

## Files

- **Orecoin_Logo_Coin_Front.svg/png**: The primary face of the coin featuring the Ziggurat of Ur.
- **Orecoin_Logo_Coin_Back.svg/png**: The reverse face featuring King Gilgamesh.
- **Orecoin_Logo_Full.svg/png**: The full logo (Front Face).
- **Orecoin_Logo_Favicon.svg/png**: Simplified icon for browser tabs and small screens.
- **Orecoin_Logo_Monochrome.png**: Black and white version for print.
- **Orecoin_Logo_Variations.png**: Tinted variation example.

## Source

The source of truth is the `.svg` files. If you need to regenerate the PNGs at different sizes, you can use tools like `sharp`, `imagemagick`, or `Adobe Illustrator`.

## Colors

- Gold: Gradient (#FEDC56 -> #D4AF37 -> #996515)
- Bronze: Gradient (#E8A870 -> #CD7F32 -> #8B4513)
- Text Color: #5A3A20
